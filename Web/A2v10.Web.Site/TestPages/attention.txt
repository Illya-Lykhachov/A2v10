
Таблицы
==============
1. Если border-collapse = collapsed и <thead> стоит после <tbody>, то в 
Chrome съедаются рамки для td. (а для th - остаются).

Разное
==============
1. В Chrome (или в Vue) id элементов не могут начинаться с цифры!


Как что работает
================
1. CollectionView + DataGrid. RunAt указывает режим работы. 
Может быть: Client(default) | Server / ServerUrl
Client - обработка на клиенте
Server - обработка на сервере, UrlQuery не используется. 
	(иницируется событие localQueryChange корневого контроллера)
ServerUrl - обработка на сервере, используется UrlQuery
	(новый query передается в routex)


2. DataGrid может быть как внутри CollectionView, так и нет. Если нет, то 
группировка и фильтрация не поддерживается. 

Сортировка будет выполняться локально внутри самого DataGrid. 
Сама сортировка в DataGrid включается флажком Sort:Boolean.
